# Crestkey

### What's Crestkey?
A fully offline, AES-256 encrypted password vault built as a WinForms .NET Framework desktop app. Single encrypted file on disk, master password derives the key via PBKDF2/Argon2, nothing ever leaves the machine.

### Who's it for?
- Developers and power users who are skeptical of cloud vaults (Bitwarden breach, LastPass disaster, etc.)
- People who want full auditability - open source, they can read every line
- Privacy-conscious users who don't want a company holding their encrypted blob
- Anyone already burned by a subscription-based password manager
